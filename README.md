# 💰 FinansTakip - Kişisel Finans Yönetim Sistemi

Java Swing kullanılarak geliştirilmiş, kullanıcıların gelir ve giderlerini takip etmelerini sağlayan masaüstü uygulaması.

## 🚀 Özellikler
* **Gelir/Gider Kaydı:** Günlük finansal hareketlerin kategorize edilerek kaydedilmesi.
* **Veri Saklama:** Girilen verilerin `veriler.txt` dosyası üzerinde kalıcı olarak depolanması.
* **Grafiksel Analiz:** Finansal verilerin görselleştirilmesi için özelleştirilmiş grafik panelleri.
* **Kullanıcı Dostu Arayüz:** Java Swing kütüphanesi ile tasarlanmış sezgisel UI.

## 🛠️ Kullanılan Teknolojiler
* **Dil:** Java
* **Arayüz:** Java Swing & AWT
* **Veri Yönetimi:** File I/O (Dosya İşlemleri)

## 📁 Proje Yapısı
* `FinansSwing.java`: Ana uygulama mantığı ve arayüz bileşenleri.
* `FinansSwing$GrafikPanel.class`: Verilerin görselleştirildiği grafik arayüzü.
* `veriler.txt`: Uygulamanın veritabanı olarak kullandığı metin dosyası.
